AudiHealth: Vocal Disorder Detection using Deep Learning

- Built a CNN-based vocal disorder classification system using Mel-Spectrogram audio features.
- Applied Librosa preprocessing, audio augmentation techniques, and SMOTE-based class balancing.
- Integrated Grad-CAM for model interpretability and explainable predictions.
- Achieved 84.69% accuracy and 0.85 macro F1-score for Healthy, Laryngitis, and Vocal Polyp classification.
